# PNG to BTX Converter for BlackRussia

Конвертер изображений между форматами PNG и BTX (специальный формат для игры BlackRussia).

## Возможности
- Конвертация PNG → BTX
- Конвертация BTX → PNG
- Пакетная обработка ZIP-архивов:
  - ZIP с PNG → ZIP с BTX
  - ZIP с BTX → ZIP с PNG
